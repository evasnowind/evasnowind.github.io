# git使用技巧-撤销修改

git撤销修改

## 丢弃工作区的修改

```
git checkout  --文件名称
```

## 把暂存区的修改撤销掉（unstage），重新放回工作区

```
git reset HEAD 文件名称
```

## 已经commit到版本库，想撤回本次commit

前提：没有推送到远端

```
git reset --hard HEAD^
```

或者

```
git reset --hard commit_id
```
