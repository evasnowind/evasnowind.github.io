# git使用技巧-快速回退远端分支到某个指定版本

先使用

```
git log
```

获取想要回退版本的commit id，假定此处为`82d9cab12731870bd04327464e501479da3a87e2`

然后通过

```
git reset 82d9cab12731870bd04327464e501479da3a87e2
```

将git本地版本回滚到指定commit。

然后通过

```
git push -f
```

将本地更新强制推送到远端。
